backend/
├── server.js               # Startpunkt der App (Express-Server)
├── config/
│   └── db.js               # Verbindung zu MongoDB
├── routes/
│   └── linkRoutes.js       # Endpunkte für Links
├── controllers/
│   └── linkController.js   # Logik für Link-Endpunkte
├── models/
│   └── Link.js             # Mongoose Schema & Model
├── .env                    # Umgebungsvariablen (z. B. MONGO_URI)
├── package.json            # Projektkonfiguration (Node.js)
└── node_modules/           # Installierte Pakete


| Ordner/Datei       | Aufgabe                                                                         |
| ------------------ | ------------------------------------------------------------------------------- |
| `server.js`        | Express-App initialisieren, Server starten, Middleware laden, Routen einbinden  |
| `config/db.js`     | Stellt die Verbindung mit MongoDB her (z. B. mit `mongoose.connect()`)          |
| `routes/*.js`      | Definiert die REST-Endpunkte (z. B. `GET /api/links`)                           |
| `controllers/*.js` | Enthält die eigentliche Logik z. B. zum Abruf, Speichern oder Löschen von Daten |
| `models/*.js`      | Mongoose-Schema und -Model (z. B. wie ein Link aufgebaut ist)                   |
| `.env`             | Speichert geheime und veränderbare Daten wie `MONGO_URI` und `PORT`             |
